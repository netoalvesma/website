Meus Projetos

Sistema de Gerenciamento de Tarefas

Desenvolvi um sistema de gerenciamento de tarefas voltado para pequenas equipes, permitindo o acompanhamento eficiente do progresso dos projetos. A aplicação permite criar, editar e organizar tarefas, atribuí-las a membros da equipe e definir prazos. Utilizei tecnologias como JavaScript, Node.js e MongoDB para o backend, garantindo um sistema escalável e responsivo, enquanto no frontend, implementei uma interface intuitiva usando React, facilitando a experiência do usuário. O projeto também conta com funcionalidades de autenticação e autorização para garantir a segurança dos dados.

Plataforma de Controle Financeiro Pessoal

Desenvolvi uma plataforma web para controle financeiro pessoal que permite aos usuários gerenciar suas finanças de forma prática e intuitiva. A aplicação possibilita o cadastro de receitas e despesas, categorização de gastos, e geração de relatórios mensais para análise financeira. Utilizei Python com Django no backend para lidar com a lógica de negócios e banco de dados relacional, e no frontend implementei uma interface responsiva com HTML, CSS e JavaScript. A aplicação também oferece gráficos dinâmicos para visualização das finanças e recursos de exportação de dados para planilhas.

Aplicativo de Monitoramento de Hábitos

Desenvolvi um aplicativo mobile para monitoramento de hábitos diários, com foco em ajudar os usuários a criar e manter rotinas saudáveis. O app permite o cadastro de hábitos, como leitura, exercícios e meditação, além de enviar lembretes diários para incentivar a continuidade. Utilizei Flutter para criar um aplicativo multiplataforma (iOS e Android), integrando Firebase para o armazenamento de dados em tempo real. A interface do usuário foi projetada para ser simples e intuitiva, permitindo uma fácil visualização do progresso e estatísticas semanais de cada hábito.

E-commerce para Produtos Artesanais

Desenvolvi uma plataforma de e-commerce voltada para a venda de produtos artesanais. O sistema permite a criação de contas de vendedores e compradores, gerenciamento de estoque, processamento de pedidos e integração com gateways de pagamento. A aplicação foi desenvolvida utilizando Node.js e Express no backend, com um banco de dados MySQL para o gerenciamento dos produtos e transações. No frontend, utilizei React para criar uma interface responsiva e moderna, proporcionando uma experiência de compra intuitiva. O sistema também inclui funcionalidades como carrinho de compras, avaliações de produtos e um painel de administração para os vendedores acompanharem suas vendas e pedidos.
